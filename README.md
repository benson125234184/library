# Nuxt Shadcn 圖書館管理系統

這是一個使用 Nuxt.js 和 Shadcn UI 開發的圖書館管理系統前端介面。

## 功能特點

- 📚 借閱紀錄管理
- 🔄 續借功能
- 📱 響應式設計
- 📊 表格/網格雙視圖模式
- 🔍 多欄位排序
- 📄 分頁功能

## 技術棧

- Nuxt.js
- Vue 3
- Shadcn UI
- Tailwind CSS

## 安裝步驟

1. 克隆專案
```bash
git clone [your-repository-url]
cd nuxt-shadcn-dashboard
```

2. 安裝依賴
```bash
npm install
# 或
yarn install
```

3. 啟動開發伺服器
```bash
npm run dev
# 或
yarn dev
```

4. 開啟瀏覽器訪問 `http://localhost:3000`

## 使用說明

- 可以在表格和網格視圖之間切換
- 點擊表格標題可以進行排序
- 使用頁面底部的分頁控制器瀏覽更多記錄
- 點擊「續借」按鈕可以延長借閱期限
